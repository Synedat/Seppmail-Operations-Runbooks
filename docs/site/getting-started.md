# Getting started

## First steps

- Read the repository overview and the quickstart section in `README.md`
- Review prerequisites and permissions in `docs/RBAC-AND-PERMISSIONS.md`
- Work through the examples in `examples/`
- Use the demo scenarios to validate your understanding

## Practical adoption path

- Start in a lab or non-production tenant/subscription
- Baseline your current state before applying changes
- Record assumptions, evidence and rollback options
- Promote stable patterns into shared runbooks or CI pipelines
