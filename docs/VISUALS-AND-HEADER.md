## Screenshot approach

If you later add real product or portal screenshots, prefer these rules:

- sanitize tenant names, hostnames and identities
- crop tightly to the operational value
- annotate decision points instead of sharing full consoles
- avoid showing secrets, serials or sensitive routing data
- export stable PNG versions for long-lived documentation
