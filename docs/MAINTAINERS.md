# Maintainers and Ownership

## Maintainer model

This repository is maintained by Synedat Group GmbH.

## Suggested roles

- Repository owner: strategic direction, release approval, public messaging
- Technical maintainer: code, docs, examples, workflows
- Security reviewer: disclosure handling, hardening reviews, dependency hygiene
- Community maintainer: issue triage, discussions, release notes, examples backlog

## Review cadence

- weekly issue and PR triage
- monthly documentation refresh
- quarterly roadmap and example review

## Public contribution guardrails

Review external contributions for:

- licensing clarity
- no secrets or internal-only information
- technically reproducible examples
- alignment with README positioning and support boundaries
